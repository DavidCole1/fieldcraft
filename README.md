# Anchor Digital

Marketing site for Anchor Digital — websites for local businesses, built with AI, based in Alexandria, VA.

## Structure

- `index.html` — the live site (hero, portfolio, pricing, FAQ, contact)
- `samples/` — three sample client builds shown as live previews on the homepage:
  - `rootline-landscaping.html`
  - `basin-and-blade-salon.html`
  - `iron-bay-auto.html`

## Deploying

This is a static site with no build step. To deploy on Vercel:

1. Push this folder to a GitHub repository.
2. In Vercel, click **Add New Project** → **Import** the repository.
3. Leave all settings as default (no framework, no build command) and click **Deploy**.

No environment variables or configuration needed.
